# WingspanSettlement

展翅翱翔（Wingspan）桌遊計分表工具 - 一個簡潔易用的線上計分器。

## 📖 專案簡介

WingspanSettlement 是一個專為《展翅翱翔》（Wingspan）桌遊設計的計分工具。透過簡潔的網頁介面，幫助玩家快速記錄和計算遊戲分數，無需紙筆即可完成計分。

**最輕便的設計**：本專案採用極簡架構，整個專案僅包含單一 HTML 檔案，程式碼不到 200 行，無需任何外部依賴或建置步驟，可直接在瀏覽器中開啟使用。

## ✨ 功能特點

- 🎯 **即時自動計算**：輸入分數後自動計算每位玩家的總分
- 👥 **多玩家支援**：支援最多 5 位玩家同時計分
- 📊 **完整計分項目**：包含所有計分項
  - 鳥類卡
  - 獎勵卡
  - 鳥蛋
  - 儲存的食物
  - 塞入的卡牌
- 📱 **響應式設計**：適配各種螢幕尺寸，支援橫向滾動
- 🔒 **固定首列**：計分項目名稱固定顯示，方便查看
- ✏️ **直接編輯**：點擊儲存格即可輸入分數，操作簡單直觀
- ⚡ **極致輕量**：單一檔案，不到 200 行程式碼，載入快速

## 🚀 使用方法

1. 直接在瀏覽器中開啟 `index.html` 檔案
2. 點擊表格中的儲存格，輸入每位玩家各項得分
3. 系統會自動計算並顯示每位玩家的總分
4. 無需儲存，重新整理頁面即可重置

## 🛠️ 技術棧

- **HTML5**：頁面結構
- **CSS3**：樣式設計和響應式佈局
- **JavaScript (Vanilla)**：即時計算邏輯

## 📝 使用說明

- 所有可編輯的儲存格（白色背景）都可以直接點擊輸入數字
- 輸入時儲存格會顯示綠色邊框提示
- 總分行（灰色背景）會自動計算，不可編輯
- 第一列（項目名稱）在橫向滾動時會保持固定顯示

## 🎮 關於展翅翱翔

《展翅翱翔》（Wingspan）是一款以鳥類為主題的引擎構築類桌遊，玩家需要透過收集鳥類卡牌、產蛋、獲取食物等方式來獲得分數。本工具幫助玩家在遊戲結束時快速準確地計算最終得分。
